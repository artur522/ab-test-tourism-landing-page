# A/B тестирование посадочной страницы турагентства

## Описание проекта
Анализ результатов A/B теста двух вариантов посадочной страницы туристической компании по метрикам конверсии и среднего чека.

## Ключевые выводы
- **Конверсия:** статистически значимой разницы нет (p-value = 0.2394)
- **Средний чек:** вариант B показывает значительное увеличение на ~7,600 рублей (p-value ≈ 0.0000)
- **Рекомендация:** вариант B приносит на 786 руб. больше выручки с посетителя

## Структура
- `ab-test-tourism-landing-page.ipynb` — полный анализ в Jupyter Notebook
- `data/ab_data_tourist.csv` — исходные данные
